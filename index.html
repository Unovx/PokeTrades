What's New: October 2021

Another option was added which I think you'll love, the option to see sprites instead of the normal Home artwork for gens 1-5. This option is off by default, but once turned on, you will see older sprites depending on the game they were obtained in. If a sprite does not exist for it, the default Home artwork is used.

What's New: September 2021

There is now an option to see the Dex Number of the Pokemon in the Selection Area. Additionally the Advanced Preview option also now shows the Pokemon Name and Typing to fill the empty space.

The way to create and move Bunches has been improved. Now you simply need to click on the buttons above either the For Trade or Looking For Bunches.

What's New: August 2021

The default preview settings have now changed. You know see more information at a glance, but this can still be toggled to show less options. Like it used to.

Importing is now also fixed, sorry to the people that tried to previously import. I've also made it easier to import without having to change your tradesheet to be exactly how PokeTrades wants, with placeholder values when needed.

Hidden Power support is here! Previously, you would have had to manually click on the Hidden Power dropdown, but now it is automatic when toggling IVs! You can still change it manually however, and if Gigantamax is selected, it won't force the dropdown to any Hidden Power. The dropdown also won't change if an IV is X or HT.

FAQs

Q: None of the below answer my question, where can I get addtional help?

A: Click on the discord button below to join the discord and speak to me personally.

Q: What is PokeTrades?

A: PokeTrades was designed to be a better alternative to options such as Google Sheets. PokeTrades allows you to see all information without scrolling on Desktop, is actually usable on Mobile, and comes with many features from filter options to storing multiple viewings to seeing proof without having to open another tab.

Q: Does this work on Mobile?

A: Yes. There is a mobile view allowing you to clearly see everything.

Q: How do I re-arrange my Pokemon or Bunches?

A: To do this, you must be signed in. Then after clicking on any Bunch to open it, you will be able to re-arrange any Bunch or any Pokemon inside that Bunch

Q: How do I copy a Pokemon?

A: When you click on the Move/Copy button, click on the same Pokemon twice and it will create a copy right after it.

Q: How do I create a Bunch?

A: Click on the create button in the selection after and then click on Edit Bunches.
Importing your TradeSheet

To get started, you will first need to sign into your account, then in the Login Area, you should see the option to import your TradeSheet. Please make sure it is exported from Google Sheets as a "CSV" file beforehand. When importing into PokeTrades. This can be done with or without Headings, also note that when importing, natures can be simply the name without anything after as PokeTrades will format accordingly.

Please note that the order MUST BE CORRECT when importing, if you do not know what this order is, then please first export your TradeSheet with the "Export" Button, as when an account is first created, there are initial creation mons, which along with the Headings will show what the order is.

There are also two more important things to keep in mind. The first is that if you give the Pokemon in the TradeSheet a bunch name you have not yet created, you must create the bunch name to see the pokemon in that bunch area. The trade_option Column MUST BE SET TO "For Trade" OR "Looking For", IF IT IS NONE OF THESE, YOU WILL NOT BE ABLE TO SEE OR MODIFY IT AT ALL!

If a Pokemon shows up with a Fennel Icon instead of a Pokemon Icon or fails to show up at all, then that Pokemon's image either can't be found or its an invalid combination. An example of this would be "Alcremie". While this is a Pokemon, Alcremie has many different forms, so the current form must be typed in to bring up the right Image. Click on the Pokemon and then on the "Modify" Button to enter the Creation Area and make the necessary changes and then click "Create/Update" to save the changes.

Searching TradeShops, Custom Message, Preview Icons, Additional Help To search a TradeShop, first type in the User ID of the TradeShop you want to view. You can then click on either the For Trade Image (Corless) or the Looking For Image (Aarune) to view the respective areas. If you are signed in, then your User ID will be available right underneath the InputField.

If you are signed in, when you type in your User ID in the InputField, you will be able to add a custom message by modifying the text area below. To save any changes to it, just click outside of the text area.

Preview Settings now exist to allow the user to see certain information such as Ball or IVs without having to click on the Pokemon. This can be accessed by clicking on the Klang Icon and going to Preview Settings where they can then be toggled. By default, Ball and IV previews are on. The colour coding for the IVs are:

31 IV: Blue
30 IV: Green
0 IV: Red
1 IV: Pink
HT: Yellow
Anything else: Grey

Should you need more help outside of these tutorials or if you just want to talk about PokeTrades or other things Pokemon, you can head over to the discord link in the main menu.
Creating an Account, Signing In, Changing Username and Password To register an account, first click on the Klang Icon in the Main Menu. Then click the "Register" Button. Currently, usernames are limited to 15 characters and Passwords to 50 characters.

After Signing in, You can log out or change your Username or Password in the same area. To change your login details. Click on the "Change Username or Password Button". Keep in mind that you cannot change both Username AND Password at the SAME time. This has to be done individually and duplicate Usernames are not allowed.

Only when signed in, are more options available to you such as seeing both Public and Private Pokemon on your TradeShops, being able to create Pokemon and having the option to modify them.

Once signed in, your User ID will be displayed in the Main Menu for ease of access and to share with others.
Bunches, Viewing Pokemon, Re-ordering Bunches/Pokemon, Copying Pokemon Bunches are essentially folders or tabs that contain a group of Pokemon with a small description. Clicking on a Bunch will open the relevant Pokemon inside. To view a Pokemon, click on one and you will be taken to a viewing area which shows the Pokemon inside.

Bunches and Pokemon can be re-ordered. To do this, click on the Move Button and then Click on a Bunch/Pokemon and you should see all other Bunches/Pokemon grey out, then just click on another Bunch/Pokemon you want it to be in front of. Keep in mind that the "All Pokemon" bunch cannot be re-ordered.

Pokemon can also be copied by clicking on themselves instead of another Pokemon and the new copy will appear after the original. If you have clicked on a Bunch/Pokemon but do not want to copy or move, you can cancel this action by pressing the Move Button now which will show cancel instead. Remember that to re-order or copy Bunches/Pokemon, you must be signed into your account.
Viewing Details, Modifying/Deleting Pokemon, Adding Pokemon from other TradeShops After Clicking on a Pokemon to view, this area will open up allowing you to see details about the Pokemon. Should information be uneccessary to show, it will not be displayed. An example of this would be if the Pokemon had no Nickname, then the Nickname text would be hidden.

The list of pokemon seen at the bottom are the Pokemon that the User is currently Looking For. If you are signed in and have a pokemon they want, then that Pokemon will not be greyed out, to show you have something preferred. Clicking on those Pokemon will bring up their details.

If this is your TradeShop, then you are able to click the "Modify" and "Delete" Buttons. allowing you to modify and delete your pokemon accordingly. If this is not your TradeShop but you are signed in, you can use the "Add" Button to add a copy of the Pokemon you are currently viewing to your TradeShop.
Creating Pokemon, Pokemon Validation, Private/Public Pokemon Clicking on the "Create" Button in the Selection Area or the "Modify" Button in the Viewing Area will take you to the Creation Area. If accessed via the "Modify" Button, the details will be set to what the Pokemon currently has.

Via Looking For, there is more freedom in what can be selected such as Any Ball or Any Gender. For Trade has less freedom as the User will be detailing what the Pokemon has rather than what they want. Also a Pokemon created as Private is visible only to the User when they are signed in.
Creating Bunches/Icons, Updating/Removing Bunches, Renaming Bunches Currently, to create a Bunch, you must be signed in and in the Creation Area for creating a Pokemon. Then press the "Edit Bunches" Button to go to the Bunch Area.

To create a Bunch, an Icon must first be chosen. In addition to having all Pokemon available as Icons, forms that can't usually be chosen for a Pokemon such as Megas and GMax, all Pokeballs and the Egg Icon are available. After selecting the Icon, type a Bunch name into the InputField below and hit the "Add/Update Button" to add the new Bunch name. To delete a Bunch name, simply type in the name and hit the "Remove Bunch" Button to remove. Icon exclusives like the ones above do not need a gender set.

To rename a Bunch and give all the Pokemon with the old Bunch name the new one, type in the new name on the first InputField on the side. Then select the Bunch name to replace and select the "Rename Bunch" Button.
Filtering Options, Private/Public Display Option, Filter Indication To filter a Pokemon by name, you can simply type in the Pokemon name in the Selection Area. However, for advanced filtering, click on the "Filters" Button in the Selection Area to open up the Filter Area.

In this area you are able to filter Pokemon in a multitude of ways and all of these filtering options can be used at once. However, only the Owner of the TradeShop can use the filter to choose between Private and Public Pokemon. Please note that if an option starts with an "(Any)" in brackets, then that filter option will act like it has not been given any input and so will not be involved in filtering the Pokemon. The same also applies to X for IVs/EVs

If one or more of the advanced filters are in use, then the "Filters" Button will turn purple to let you know that the Pokemon are currently being filtered.
